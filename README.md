# Project Starter Template

## Tech stacks
- Nuxt
- Tailwind
- FormKit
- PrimeVue

## Requirements
- Node.js - `v18.0.0` or newer

## Setup

Make sure to install the dependencies:

```bash

pnpm install

```

## Development

```bash

# all package
pnpm dev

# app only
pnpm app dev

```

## Production

```bash

# all package
pnpm build

# app onky
pnpm app build

```
